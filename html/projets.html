Python Project

Octobre 2021

×

Pour ce projet nous devions réaliser le mythique jeu du casse-brique. Le projet à été réalisé en groupe en terminale spécialité Numérique et Sciences de l'Informatique.


Description :


Pour la réalisation du casse-brique nous avons utilisé python pour la partie algorithmie et Processing pour la partie graphique.


Lors de ce projet réalisé au lycée j'abordais pour la toute première fois le concepte de Programmation Orienté Objet. Nous avions à disposition les différentes classes qui allaient composé le jeu (brick, ball et paddle) et, avec ces classes, nous devions les appeler afin de développer un casse-brique fonctionnel avec plusieurs niveaux.
